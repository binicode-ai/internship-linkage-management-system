# Internship & Industry Linkage Management System

A modern web-based Internship & Industry Linkage Management System designed to connect students, universities, and companies.

## Features

- Student Portal
- Admin Dashboard
- Internship Posting
- Application Tracking
- Evaluation System
- Complaint & Communication System
- Responsive Modern UI

## Technologies Used

- Node.js
- Express.js
- PostgreSQL
- EJS
- CSS3
- JavaScript
